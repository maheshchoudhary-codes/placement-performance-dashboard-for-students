# Placement Performance Dashboard for Students

# Overview

I built this project to understand how different factors like academic performance, skills, and other attributes affect student placements. Instead of just looking at raw data, I focused on identifying patterns and insights that can actually help in decision-making.

The system analyzes student placement data and highlights key trends related to placement success.

# What this project does

- Analyzes student academic performance (CGPA, scores, etc.)
- Examines the impact of skills on placement outcomes
- Identifies patterns between different factors and placement success
- Visualizes data using charts for better understanding
- Helps in finding which factors matter the most for getting placed

# Key Insights

- Students with higher CGPA tend to have better placement rates  
- Technical skills play a significant role in placement success  
- Certain combinations of skills and academics increase chances of selection  

# Tech Stack

- Python  
- Pandas  
- NumPy  
- Matplotlib / Seaborn  

#How to run

1. Install dependencies:
   pip install pandas numpy matplotlib seaborn

2. Run the script:
   python placement_dashboard.py
